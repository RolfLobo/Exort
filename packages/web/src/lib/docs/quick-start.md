# Quick Start

This guide gets you from installation to uploading your first project.

## Step 1 - Install Exort

Download the latest Exort build from:

- [https://exort.dev/download](https://exort.dev/download)
- <a href={EXORT_GITHUB_RELEASES_LINK}>{EXORT_GITHUB_RELEASES_LINK}</a>

The latest public release visible on GitHub is Exort <code>v{EXORT_APP_VERSION}</code>.

## Step 2 - Open a Workspace

Open a local folder where your embedded project lives.

A workspace can be:

- An existing Arduino project
- A new empty folder
- A firmware project generated by Exort Agent
- A folder containing `.ino`, `.cpp`, `.h`, or config files

## Step 3 - Connect a Provider

Exort is powered by OpenCode. You can start with included free OpenCode models or connect your own OpenCode-compatible provider setup.

## Step 4 - Ask Exort Agent

Example prompts:

- Create a simple Arduino blink project.
- Explain this ESP32 code and tell me what each pin does.
- Add serial logging so I can debug the sensor values.
- Refactor this code into separate files.
- Find why this sketch is not compiling.

## Step 5 - Select Board and Port

Choose your target board and serial port from the app.

## Step 6 - Compile and Upload

Use Exort compile/upload controls to build the project and upload it to your device.

## Step 7 - Monitor Output

Open Serial Monitor to watch logs, boot messages, debug prints, sensor values, or calibration output.

Use Serial Plotter when your board prints numeric values that should be visualized.
